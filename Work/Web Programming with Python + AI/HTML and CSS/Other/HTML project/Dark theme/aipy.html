About the course

This course covers a large amount of material. It is meant for dedicated programmers that want to learn how to

to get a fully completed project.

  • Gemini - integration into the website

Who is this course recommended to?

  • Advanced programmers
  • Serious tech enthusiasts
  • People looking to create web-apps for a business

Prerequisites
