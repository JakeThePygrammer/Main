Codemonkey course

About the course

The Codemonkey course is a great option for kids that want to start their coding journey early. It
is a great choice for those who would like to learn how to solve problems using code in a gamified way,
with interesting visuals..

Skills acquired

You will learn the basics of coding in Python over 25 hours.

Who is this course recommended to?

  • Young programmers
  • Child tech enthusiasts
  • Kids looking to expand their knowledge

Prerequisites

  • Basic understanding of using a computer
  • Basic knowledge of English

Certificates

By passing the course, you acquire a course attendance certificate issued by Semos Education.

Payment

The course in total is 7000 denars. If you are a recurring customer, you get 10% off.
