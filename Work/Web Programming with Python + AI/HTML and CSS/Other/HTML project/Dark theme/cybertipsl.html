Cyber and safety tips

27.02.2026 - When setting up an account, make sure to set up 2-factor authentication and set up a strong password to make sure that
you can't get hacked! Make sure to include many special characters in your password, to ensure it's hard to crack.

26.02.2026 - Make sure to check your passwords regularly! Data breaches happen very often!

25.02.2026 - When using public or even private Wi-Fi, utilizing a VPN(virtual private network) can help protect sensitive data being
sent from your computer.

24.02.2026 - If you are visiting a site where you must enter sensitive information, make sure that it is the right site! You may be falling for a scam!
Carefully inspect the domain before sharing personal information!

23.02.2026 - Make sure to back up your data! You never know what might happen!

22.02.2026 - Be cautious with AI, it may seem right, but not always!

21.02.2026 - Never give personal information on the internet! You never know who's on the other side of the screen!

20.02.2026 - If you get a suspicious email, don't pay attention to it! Report it using the app's in-built report feature!

19.02.2026 - Don't download shady files! No, you're not getting free Robux.

18.02.2026 - Change passwords regularly! Make sure that they are always up to date!

17.02.2026 - When downloading a new app, make sure to see what permissions it uses. Make sure that they match the app's function.

A good password must have :

  • 8 characters or more
  • A combination of numbers, letters and symbols
  • Not be associated to you(birthdays, name/surname)
  • Unique - don't use the same password on multiple sites
