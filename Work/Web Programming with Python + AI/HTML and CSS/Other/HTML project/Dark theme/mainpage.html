About us

Semos education is a premium educational institution located in Skopje, which has courses for children and adults alike.
Semos education started in 1995 in Macedonia. We offer a variety of programming and media editing courses. Our mission is to
educate as many people as possible on advanced computer literacy, which can be seen in our mission statement "Empowering Digital Minds".

Cyber tip of the day :

When setting up an account, make sure to set up 2-factor authentication and set up a strong password to make sure that
you can't get hacked! Make sure to include many special characters in your password, to ensure it's hard to crack.
